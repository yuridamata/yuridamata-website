# Yuri da Mata - Main Website

This is the main website for yuridamata.com.br built with Vite, React, and Tailwind CSS.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
